프로젝트 소개

매일 변동되는 식자재 단가의 수기 입력과 원가 계산을 AI로 자동화하여, 외식업 사장님에게 직관적인 마진율 대시보드를 제공하는 B2B SaaS를 구축합니다.

핵심 요구사항 분석

  • AI Vision OCR & 자동 재학습 — 네이티브 카메라 촬영·업로드, GPT-4o Vision/Claude API 기반 품목명·수량·단가·공급업체 JSON 추출, 외식업 품목명 사전 기반 이표기 자동 매핑, 사용자 교정 UI 및 교정 데이터 활용 재학습 파이프라인
  • 3단계 원가 계산 엔진 — 식자재→프렙→메뉴 계층형 DB, 식자재 단가 변경 시 연관 메뉴 원가율 연쇄 자동 업데이트, 목표 원가율 초과 시 실시간 푸시 알림, 메뉴별 원가율/마진율 차트, 월 고정비·인건비 기반 수익 구조 진단
  • 멀티플랫폼 프론트엔드 — React Native 사용자 앱, React 웹 대시보드, Supabase Auth 회원가입/로그인 및 개인정보보호법 기반 동의 UI
  • 최고 관리자 백오피스 — 전체 가입 식당 관리, 글로벌 품목 사전 마스터 데이터 관리, 식당별 API 호출 통계·에러 로그 모니터링
  • 비기능 요구사항 — Vision API 응답 지연 대응 로딩 UX, Supabase RLS 기반 식당 간 데이터 완전 격리

제안 포인트

  • 정확도와 자가 개선이 핵심인 OCR 파이프라인 — Vision API 추출 결과를 외식업 전용 품목명 사전과 매핑하여 이표기를 흡수하고, 사용자 교정 데이터를 정답 데이터셋으로 축적해 프롬프트·매핑 규칙을 지속 보정하는 재학습 루프를 설계합니다.
  • 연쇄 업데이트가 보장되는 3단계 원가 엔진 — 계층형 데이터 모델과 집계 트리거로 특정 식자재 단가가 바뀌면 이를 사용하는 프렙·메뉴 원가율이 자동 재계산되며, 목표 원가율 초과 시점에 실시간 푸시로 사장님께 즉시 알립니다.
  • 완벽한 멀티테넌트 데이터 격리 — Supabase RLS 정책으로 식당 단위 행 수준 격리를 구현하여 식당 간 데이터 노출을 원천 차단합니다. (유사 멀티테넌트 SaaS 다수 구축 경험)
  • iOS·Android·웹·관리자 일괄 제공 — 단일 데이터 모델 위에서 React Native 앱과 React 웹·관리자 백오피스를 통합 개발하여 일관된 UX와 유지보수성을 확보합니다.
  • 지연을 자연스럽게 흡수하는 UX — Vision API 호출은 비동기 처리하고, 업로드→분석→교정 흐름에 스켈레톤·진행률 로딩을 적용하여 체감 대기 시간을 최소화합니다.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

외식업 운영에서 매일 변동되는 식자재 단가를 수기로 입력하고 메뉴 원가를 계산하는 작업은 번거롭고 오류가 잦습니다. 본 프로젝트는 거래명세서 촬영 한 번으로 품목·단가를 자동 추출하고, 식자재에서 프렙·메뉴로 이어지는 3단계 원가 계산을 자동화하여, 외식업 사장님에게 직관적인 마진율 대시보드를 제공하는 B2B SaaS를 구축하는 것을 목표로 합니다.

2. 사용자 정의

사용자 유형설명주요 기능
식당 사장/매니저매장 운영 및 원가 관리 주체명세서 촬영·교정, 메뉴 원가·마진 확인, 목표 원가율 설정, 푸시 알림 수신
실무 직원입고/식자재 관리 담당명세서 업로드, 인식 결과 교정
최고 관리자(운영사)플랫폼 전체 운영식당 관리, 글로벌 품목 사전 마스터 관리, API 호출 통계·에러 로그 모니터링

3. 기능 요구사항

3.1 AI Vision OCR & 자동 재학습

  • [필수] 모바일 앱 네이티브 카메라 연동 및 거래명세서 촬영·업로드
  • [필수] GPT-4o Vision 또는 Claude API 연동 — 품목명·수량·단가·공급업체 JSON 추출
  • [필수] 외식업 전용 품목명 사전 기반 이표기 자동 매핑
  • [필수] 사용자 오인식 수동 교정 UI
  • [필수] 교정 데이터를 활용한 AI 자동 재학습 파이프라인 연동

3.2 3단계 원가 계산 엔진 & 대시보드

  • [필수] 식자재→프렙→메뉴 3단계 계층 구조 DB 설계
  • [필수] 식자재 단가 변경 시 연관 메뉴 원가율 연쇄 자동 업데이트
  • [필수] 목표 원가율 설정 및 초과 시 실시간 앱 푸시 알림
  • [필수] 메뉴별 원가율·마진율 실시간 표시 및 목표 대비 비교 차트
  • [필수] 월 고정비·인건비 입력 기반 전체 수익 구조 진단 화면

3.3 프론트엔드 & 최고 관리자 백오피스

  • [필수] Supabase Auth 회원가입/로그인 및 개인정보보호법 기반 동의 UI
  • [필수] React Native 기반 사용자용 모바일 앱 (iOS/Android)
  • [필수] React 기반 사용자용 PC 웹 대시보드
  • [필수] 전체 가입 식당 관리, 글로벌 품목 사전 마스터 관리, 식당별 API 호출 통계·에러 로그 모니터링 관리자 웹
